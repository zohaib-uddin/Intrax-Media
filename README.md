# Intrax Media – Agency Portfolio Website

Intrax Media is a modern and responsive agency portfolio website designed to showcase digital services, projects, and company information in a clean and professional way. The website highlights the agency’s capabilities, services, and portfolio while providing an engaging user experience for potential clients.

* Contact section for potential clients
* Fast loading and optimized layout
* Mobile-friendly design

## 🛠️ Technologies Used

This project was built using a modern full-stack web development stack to ensure performance, scalability, and a smooth user experience.

### Frontend

* **React.js** – Component-based UI development
* **TypeScript** – Type-safe JavaScript for better code reliability
* **HTML5 & CSS3** – Structure and styling
* **Responsive Design** – Mobile-friendly layout

### Backend

* **Node.js** – Server-side runtime environment
* **Supabase** – Backend-as-a-Service for database, authentication, and API management

### Integrations

* **Calendly** – Appointment scheduling integration for client bookings

### Development Stack

* **MERN Stack Architecture**
* **Git & GitHub** – Version control and project management

- Integrated booking system via Calendly


```bash

```

3. Install dependencies


```bash

```
